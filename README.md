# Roomba Cat Stalker

Roomba Cat Stalker is my modification of a half broken roomba to allow me to stalk my cat at home. 

## What is Does

The Roomba Cat Stalker lets me follow my cat around my apartment via the web. As the name hints at, the Roomba Cat Stalker is a embedded device mounted on a roomba with a USB camera attached. Maybe more to come, water gun? nerf gun? cat toy on robotic arm?
